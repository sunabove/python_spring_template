  • Pico Version 2 출시 *

    • 2024년 8월 8일에 Raspberry Pi Pico V2가 출시되었습니다.



    • 마이크로 컨트롤러 비교
      - Raspberry Pi Pico는 RP2040 칩으로 구동됩니다.
      - Raspberry Pi Pico 2는 RP2350의 칩을 사용합니다.
    • 프로세서 비교
      - Raspberry Pi Pico : 최대 133MHz 듀얼코어 Arm Cortex M0+ 프로세서를 탑재
      - Raspberry Pi Pico 2 : 최대 150MHz 듀얼 Cortex-M33 또는 Hazard3 프로세서를 탑재
    • 메모리
      - Raspberry Pi Pico : 264KB SRAM과 2MB 온보드 플래시 메모리
      - Raspberry Pi Pico 2 : 520KB SRAM과 4MB의 온보드 플래시 메모리
    • GPIO
      - Raspberry Pi Pico : ADC에 사용 가능한 3개를 포함하여 총 26개의 다기능 GPIO 핀과 2× SPI, 2× I2C, 2× UART, 3× 12비트 ADC, 16× 제어 가능한 PWM 채널을 제공
      - Raspberry Pi Pico 2 : 26개의 다기능 GPIO 핀을 유지하지만 3개의 12비트 500ksps ADC를 포함하고 PWM 채널을 24개로 확장
      또한 2개의 SPI, 2개의 I2C, 2개의 UART, 4개의 알람이 있는 2개의 타이머, 1개의 AON 타이머 인터페이스 제공

    - 2024-08-29

  • Pico 펌웨어 v1.23.0 출시 *

    • 동적 USB 장치에 대한 지원을 추가하고, 새로운 모듈들이 추가되었습니다.
    • openamp, tls웹 vfs어셈블리 포트를 완전히 개선하여 JavaScript와 Python 간의 프록시를 추가하였습니다.
    • 고정된 모듈에 대해 코드 크기를 최적화하였습니다. 그 밖에도 많은 개선 사항과 버그 수정이 이루어졌습니다.
    • 많은 설계, 개발 및 테스트를 거쳐 MicroPython은 이제 사용자 정의 USB 장치를 완벽하게 지원합니다.
    • 인터페이스는 새로운 machine.USBDevice클래스를 통해 제공되며, 이를 통해 사용자는 USB 설명자를 지정하고 USB 엔드포인트 전송을 위한 Python 콜백을 구현할 수 있습니다.
    • 이를 통하여 모든 USB 장치를 순수 Python으로 구현할 수 있습니다.
    • 인터페이스 machine.USBDevice는 저수준이고 완벽하지만, 키보드, 마우스, MIDI 및 직렬 CDC에 대한 예제와 함께 장치를 더 쉽게 구현할 수 있는 고수준 USB 라이브러리를 제공합니다.
    • micropython-lib 기능은 현재 rp2 및 samd 포트에서 사용할 수 있으며, 다른 포트도 나중에 제공될 예정입니다.

    - 2024-06-02

  • NumPy 2.0.0 출시 *

    • NumPy 2.0.0은 2006년 이후 첫 번째 주요 릴리스입니다.
    • 이전 릴리스 이후 11개월 간의 개발 결과이며, 1,078개의 Pull Rquest에 걸쳐 212명의 기여자가 참여하였습니다.
    • Python 및 C API에 대한 변경 사항은 물론 많은 흥미로운 기능이 포함되었습니다.
    • ABI 중단, 유형 승격 규칙 변경, 1.26.x에서 지원 중단 경고를 표시하지 않았을 수 있는 API 변경 등의 일반 마이너 릴리스에서는 발생할 수 없는 주요 변경 사항이 포함되었습니다.
